# App Booking Ticket for Football

## Introduction
App Booking Ticket for Football is an Android application that allows users to easily book football fields, manage schedules, and track their reservations online. The app helps players quickly find available fields and organize football matches with friends.

## Technologies
- Android Studio
- Java
- Firebase / REST API
- XML Layout
- Git & GitHub

## Features
- User registration and login
- Booking football fields
- View available time slots
- Schedule management
- Booking history
- Field information and details
- Search football fields
- Cancel or update bookings
- Notifications for booking confirmation
- User profile management
